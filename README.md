## Steps to start service
- docker ```docker compose up```
- db ```npm run referesh:views``` and ```npm run dev```
- api ```npm run dev```
- exchange_Manager ```npm run dev ```
- ws ```npm run dev ```
- frontend ```npm run dev```

Note: The Market Maker service is available as mm.
